# NMstream | Premium Music Experience

A modern, high-performance music downloader and player built with a sleek Glassmorphism design. NMstream offers a premium experience for searching, playing, and downloading your favorite music tracks.

## ✨ Features

- **Premium UI**: Modern dark theme with frosted glass effects and vibrant gradients.
- **Dynamic Search**: Real-time filtering of music tracks by title or artist.
- **Integrated Player**: Fully functional audio player with play/pause, seeking, and progress tracking.
- **Instant Downloads**: One-click download functionality for high-quality audio files.
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Micro-animations**: Smooth transitions and hover effects for an immersive experience.

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Newton990/music-app.git
   ```
2. **Open index.html**: Simply open the `index.html` file in any modern web browser.
3. **Start Streaming**: Use the search bar to find tracks or select from the trending section.

## 🛠️ Built With

- **HTML5**: Semantic structure and accessibility.
- **Vanilla CSS**: Custom design system with modern properties (backdrop-filter, CSS variables).
- **Vanilla JavaScript**: Lightweight and fast core logic (no frameworks required).
- **Google Fonts**: 'Outfit' for modern typography.
- **FontAwesome**: High-quality icons for controls.

## 📂 File Structure

- `index.html`: Main application structure.
- `styles.css`: Complete design system and layouts.
- `script.js`: Audio logic, search functionality, and UI interactions.
- `README.md`: Project documentation.
